# README: KMS Медицина — Платная скорая помощь

## О проекте

Корпоративный сайт для службы платной скорой помощи "КМС Медицина". Разработан на WordPress с использованием кастомной дочерней темы. Сайт представляет услуги компании, позволяет управлять SEO-метаданными из админки и содержит полную верстку лендинга с модальным окном заявки.

---

## Технологический стек

- **CMS**: WordPress 6.9.1
- **PHP**: 8.2

## Функциональность

### 1. Кастомизируемые SEO-поля

В админке WordPress при редактировании любой страницы доступны два мета-бокса:

- **SEO настройки страницы**
  - Заголовок (Title)
  - Мета-описание (Description)

- **Заголовок H1 страницы**
  - Редактируемый H1, динамически подставляется в hero-секцию

Все поля автоматически выводятся в `<head>` и переопределяют стандартные заголовки WordPress.

### 2. Модальное окно заявки

- Единая контактная форма на сайте
- Открывается по клику на любую кнопку "Оставить заявку", "Вызвать скорую", "Заказать звонок"
- Закрытие на крестик, overlay, Escape
- Кастомные стили
- Чекбокс согласия на обработку персональных данных

### 3. Плавный скролл к якорям

- Клик по пунктам меню (Услуги, Лицензии, FAQ) плавно прокручивает страницу к соответствующим секциям
- Работает как в header, так и в footer

### 4. Аккордион для FAQ

- Раскрывающиеся блоки с вопросами и ответами
- По умолчанию открыт первый вопрос

### 5. Сверстанный лендинг

Секции главной страницы:

- **Hero** — заголовок, иконки преимуществ, описание
- **Услуги** — карточки услуг с градацией светлых и темных фонов
- **Преимущества** — 7 причин выбрать компанию с нумерацией
- **Этапы работы** — алгоритм взаимодействия с клиентом
- **Виды бригад** — 4 типа медицинских бригад
- **FAQ** — 5 популярных вопросов
- **Экстренная помощь** — призыв к действию
- **Лицензии** — превью документов
- **Форма заявки** — блок с фоновым изображением
- **Футер** — контакты, навигация, соцсети, юридическая информация

```bash

```

```bash
cp .env.example .env
nano .env  # отредактируйте под свои данные
```

### 3. Запуск контейнеров

```bash
docker-compose up -d
```

- Откройте `http://localhost:8080`
- Пройдите стандартную установку WordPress
- В настройках темы активируйте дочернюю тему "kms-medicine-child"

### 5. Импорт контента

- Создайте новую страницу, назначьте ей шаблон "Главная" (front-page.php)
- В SEO-полях укажите Title и Description
- В поле H1 введите заголовок hero-секции

## SEO-настройки в админке

При редактировании любой страницы WordPress:

1. Прокрутите вниз до мета-бокса "SEO настройки страницы"
2. Заполните Title (рекомендуемая длина 50-60 символов)
3. Заполните Description (рекомендуемая длина 150-160 символов)
4. В блоке "Заголовок H1 страницы" укажите текст для hero-секции
5. Опубликуйте страницу

Все изменения сразу отображаются на сайте.

## Реализация редактируемых SEO-полей

SEO-поля реализованы с ACF

При установленном плагине Advanced Custom Fields:

1. Создана группа полей "SEO настройки" с локацией "Страница"
2. Поля: 
   - `seo_title` (текст)
   - `seo_description` (текстовое область)
3. Вывод реализован через `get_field()` в `header.php`

## Известные особенности

- В теме отключена стандартная поддержка `title-tag`, управление Title происходит через кастомные поля
- Все ссылки в меню и футере работают как якоря
- Модальное окно открывается с кнопок по сайту
- Для корректной работы плавного скролла у секций должны быть ID: #services, #licenses, #faq, #advantages, #work-types, #contact
